# Contributing

Thanks for the interest! A few notes before you open an issue or PR.

## Status

This repository is **pre-release**. No working code has been shipped yet — it is a placeholder while the design is finalized. Expect the initial implementation to be a **full rewrite** of anything currently in the tree.

## Issues

Issues are welcome, especially:
- Use-case suggestions for the planned skills
- Feedback on the design in [`DESIGN.md`](DESIGN.md)
- Pointers to prior art (other Notion integrations, MCP servers, etc.)

## Pull requests

Given the pre-release state, please open an issue to discuss before sending a PR. Code contributions will be rebased or rewritten once the MVP architecture lands.
