# notion-plugin

> 🚧 Pre-release — placeholder repository. No working code yet.

Notion integration for Claude. Automate schema setup, bulk edits, and cross-references across workspaces.

## Planned Skills

- **notion-schema-setup** [planned] — Create Notion databases with properties from a brief
- **notion-bulk-edit** [planned] — Apply transformations across many pages
- **notion-sync** [planned] — Two-way sync between Notion and local Markdown files
- **notion-cross-reference** [planned] — Link related pages automatically
- **notion-template-instantiate** [planned] — Fill templates from structured input

## Requirements (planned)

- Notion integration token (internal integration)
- [Notion SDK for JavaScript](https://github.com/makenotion/notion-sdk-js)
- Node.js 18+
